# Contributing to NAYAK

Welcome. You are building the foundation that 
the world's robots will run on.

## How to Contribute

1. Fork the repo
2. Create a branch: git checkout -b feat/your-feature
3. Build it. Test it.
4. Open a Pull Request

## What We Need

- Python engineers — core agent improvements
- AI engineers — better planning and reasoning
- Hardware hackers — robot platform support
- Writers — documentation and tutorials

## Code Standards

- Full type hints on everything
- Docstrings on all public functions
- Tests for new features
- No API keys in code ever

Apache 2.0 — your contributions are free forever.
